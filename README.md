# select-table-antd

> 


## Install

```bash
npm install --save select-table-antd
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'select-table-antd'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
